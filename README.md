## General info

Voyage is a training project (with hardcode data) of travel agency website, created for practicing markup and Scss using Webpack.

## Tools and Technologies

* HTML5.
* Scss.
* Webpack: 4.35.0.
* Bootstrap: 5.0.1.
* Air-datepicker: 2.2.3.
* Autocomplete JS: 10.2.6.
* Splide slider: 2.4.21.


## Setup

To run this project:

1. install it locally using npm:

```
$ cd ../voyage
$ npm install

```
2. add api key into the project:

- sign up and get your API key from [RAPID API](https://rapidapi.com/auth/sign-up?referral=/wirefreethought/api/geodb-cities);
- create `.env` file in root directory with content `CITIES_API_KEY=your API key`

3. start project with command:

```
$ npm start

```
